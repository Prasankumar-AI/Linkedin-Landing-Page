# LinkedIn Landing Page - Zero-Gravity Edition

A cinematic, high-performance landing page designed for **Prasan kumar Nariboina**, an Accessibility & Performance Test Specialist at IBM.

## 🚀 Vertical: AI, DevOps & Testing
This project showcases a professional transition from core Java development to specialized roles in AI (WatsonX), DevOps (Multi-Cloud), and Digital Accessibility.

## 🛠️ Approach & Logic
- **Modern Aesthetics**: Built with a "Zero-Gravity" theme, utilizing glassmorphism, glowing accents, and smooth animations to evoke a premium tech feel.
- **Performance First**: Developed using **Vanilla HTML5, CSS3, and JavaScript** to ensure ultra-fast load times and maintain a repository size well under 1MB.
- **Inclusive Design**: Following the user's role at IBM, the page structure is built with semantic HTML and accessibility-ready components.
- **AI-Driven Visuals**: Incorporated AI-generated imagery for the background and profile avatar to maintain a consistent, futuristic style.

## 🏗️ How it Works
- **Responsive Layout**: Adapts seamlessly to mobile, tablet, and desktop viewports.
- **Interactive Elements**:
  - **Particles.js**: A fixed background of moving digital starfields.
  - **Intersection Observer**: Triggers reveal animations as the user scrolls.
  - **Parallax Hero**: The central profile card reacts to mouse movements for an immersive 3D effect.
  - **Dynamic Typing**: Highlights the focus areas: AI, DevOps, and Testing.

## ☁️ Google Services Integration
- **Google Fonts**: Utilizes `Outfit` for bold headings and `Inter` for crisp body copy.
- **Google Cloud Assets**: Structure prepared for deployment on Google Cloud Run.
- **SEO Optimization**: Implemented Google-friendly meta tags, structured data placeholders, and semantic hierarchy.

## 📝 Assumptions
- **Profile Data**: Derived from the user's public LinkedIn profile as of March 2026.
- **Assets**: Visuals (hero bg, profile) are AI-generated representations to maintain the "Zero-Gravity" theme while avoiding large external dependencies.

## 📂 Repository Structure
- `index.html`: Core structure and SEO.
- `style.css`: Design system and animations.
- `script.js`: Interactive logic and particles.
- `assets/`: Optimized visual components.
